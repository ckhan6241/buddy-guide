# buddy-guide

## First time setup
```
git clone https://github.com/ckhan6241/buddy-guide.git
cd buddy-guide 
npm install 
npm start 
```
Open browser and go to http://localhost:3000/#/home-page

## Second time onwards
go to buddy-guide directory using `cd`
```
npm install
npm start
```
Open browser and go to http://localhost:3000/#/home-page
